# Herond AI Orb

**Live Preview:** [http://herond-ai-orb-pringles.netlify.app](http://herond-ai-orb-pringles.netlify.app)

A beautifully refractive, mathematically scaled, responsive, and fully customizable AI Orb visualization built with React Three Fiber, GLSL shaders, and Leva controls. 

- Features multiple aurora noise layers encapsulated inside a refractive glass lens.
- Fully responsive global canvas scaling with transparent WebGL backgrounds.
- Zero NaN glitches with isolated background refractive rendering.

## Running Locally
```bash
npm install
npm run dev
```
